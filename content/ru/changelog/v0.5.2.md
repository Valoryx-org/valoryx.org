## Интеграция биллинга

Встроенный биллинг на базе Stripe. Настраивайте тарифы, управляйте подписками и обрабатывайте обновления прямо из панели администратора.

- Интеграция вебхуков Stripe с 5 типами событий
- Управление жизненным циклом подписок (создание, обновление, отмена)
- Учёт потребления для API-вызовов и хранилища
- Генерация счетов и история платежей
- Настраиваемые лимиты тарифов для каждого рабочего пространства

## Панель аналитики

Отслеживайте эффективность документации с помощью новой панели аналитики.

- Отслеживание просмотров страниц с агрегацией по дням, неделям и месяцам
- Самые просматриваемые страницы и популярный контент
- Аналитика поисковых запросов с отслеживанием запросов без результатов
- Метрики вовлечённости пользователей (время на странице, глубина прокрутки)
- Экспорт аналитических данных в CSV

## AI-помощник для написания текстов

AI-инструменты для работы с текстом, интегрированные непосредственно в редактор.

- Подсказки по контенту и автодополнение
- Исправление грамматики и стиля
- Генерация резюме для объёмных документов
- Предложения по переводу для многоязычных рабочих пространств
- Настраиваемый LLM-провайдер (используйте свой API-ключ)

## WebAuthn и ключи доступа

Аутентификация без пароля с использованием аппаратных ключей FIDO2 и биометрии устройства.

- Регистрация нескольких ключей доступа для одного аккаунта
- Кроссплатформенная поддержка (Windows Hello, Touch ID, аппаратные ключи)
- Устойчивая к фишингу аутентификация
- Резервный вход по паролю при необходимости
- Административные настройки для принудительного использования ключей доступа

## MCP-сервер (13 инструментов)

Сервер Model Context Protocol для интеграции с AI-ассистентами.

- 13 инструментов для работы со страницами, рабочими пространствами, поиском и публикацией
- Доступ ограничен RBAC-правами подключённого пользователя
- Локальный транспорт stdio (данные не покидают вашу сеть)
- Совместимость с Claude, GPT и другими MCP-клиентами
- Полное журналирование вызовов инструментов

## Публикация документации с 5 темами

Публикуйте красивые сайты документации прямо из рабочих пространств.

- 5 встроенных тем (Minimal, Corporate, Developer, Product, API, Knowledge Base, Wiki)
- Поддержка пользовательских доменов с автоматическим TLS
- Версионирование документации с выбором ветки или тега
- Полнотекстовый поиск на опубликованных сайтах
- SEO-оптимизированный вывод с Open Graph и структурированными данными
