GitBook — одна из самых популярных платформ документации для команд разработчиков. Мы создали Valoryx, чтобы решить проблемы, которые GitBook не решает: владение git-репозиторием, self-hosting и прозрачность ценообразования.

## Опыт работы с редактором
GitBook: отличный блочный редактор, зрелые инструменты совместной работы. Valoryx: WYSIWYG на базе Tiptap, более быстрый, чистый markdown. Вердикт: GitBook зрелее; Valoryx быстрее и генерирует более чистый markdown.

## Интеграция с git
«Git Sync» в GitBook не является по-настоящему двусторонним — push из IDE вызывает конфликты слияния. Git-репозиторий не является источником истины. Valoryx: построен на git с самого начала. Документация — это и есть git-репозиторий. Content Ledger предотвращает конфликты. Вердикт: Valoryx значительно лучше.

## Self-hosting
GitBook: только облако. Valoryx: единый бинарный файл на Go, нулевые зависимости, все данные на вашем сервере. Вердикт: только Valoryx предлагает self-hosting.

## Цены
GitBook: $6,70/пользователь/месяц. Valoryx: Community (бесплатно навсегда, 5 редакторов), Team ($29/рабочая область/месяц), Business ($79/рабочая область/месяц). Вердикт: Valoryx бесплатен для небольших команд; поволичное ценообразование выгоднее для крупных команд.

## Публикуемая документация
Обе платформы предлагают профессиональный результат. GitBook отличается большей отполированностью; Valoryx обеспечивает лучшую производительность (статика) и поиск (FTS5).

## Когда выбирать GitBook
Редактирование только через браузер. Нужны зрелые инструменты совместной работы прямо сейчас. Нет требований к self-hosting. Предпочтение устоявшегося продукта.

## Когда выбирать Valoryx
Важно владение git-репозиторием. Разработчики редактируют документацию в IDE. Нужен self-hosting. Нужна бесплатная платформа для небольших команд.
