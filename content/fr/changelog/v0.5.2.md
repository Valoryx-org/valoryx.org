## Billing Integration

Stripe-powered billing is now built in. Configure plans, manage subscriptions, and handle upgrades directly from the admin dashboard.

- Stripe webhook integration with 5 event types
- Subscription lifecycle management (create, upgrade, cancel)
- Usage-based metering for API calls and storage
- Invoice generation and payment history
- Configurable plan limits per workspace

## Analytics Dashboard

Track how your documentation is performing with the new analytics dashboard.

- Page view tracking with daily, weekly, and monthly aggregation
- Most-viewed pages and trending content
- Search query analytics with zero-result tracking
- User engagement metrics (time on page, scroll depth)
- Export analytics data as CSV

## AI Writing Assist

AI-powered writing tools integrated directly into the editor.

- Content suggestions and auto-completion
- Grammar and style corrections
- Summary generation for long documents
- Translation suggestions for multilingual workspaces
- Configurable LLM provider (bring your own API key)

## WebAuthn & Passkeys

Passwordless authentication using FIDO2 security keys and platform biometrics.

- Register multiple passkeys per account
- Cross-platform support (Windows Hello, Touch ID, security keys)
- Phishing-resistant authentication
- Fallback to password when needed
- Admin controls for enforcing passkey-only login

## MCP Server (13 Tools)

Model Context Protocol server for AI assistant integration.

- 13 tools covering pages, workspaces, search, and publishing
- Scoped to the connecting user's RBAC permissions
- Local stdio transport (no data leaves your network)
- Compatible with Claude, GPT, and other MCP clients
- Full audit logging of tool invocations

## Published Docs with 5 Themes

Publish beautiful documentation sites directly from your workspaces.

- 5 built-in themes (Minimal, Corporate, Developer, Product, API, Knowledge Base, Wiki)
- Custom domain support with automatic TLS
- Versioned documentation with branch/tag selection
- Full-text search on published sites
- SEO-optimized output with Open Graph and structured data
